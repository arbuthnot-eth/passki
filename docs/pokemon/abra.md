# Abra — Superteam Agentic Engineering Grant

Abra is the Pokemon line tracking the `.SKI` team's application to Superteam's
agentic engineering grant. The grant helps builders who are already using
`solana.new` via Claude / Codex upscale their agentic engineering practice.

## Evolution path

| Form     | Levels  | Meaning                                            |
|----------|---------|----------------------------------------------------|
| Abra     | 1–15    | Drafting the application — branch, evidence, prose |
| Kadabra  | 16–35   | Application submitted, evaluation in progress      |
| Alakazam | 36+     | Grant funded, funds deployed into agent fleet      |

Each "level" is a commit or issue that moves the application forward. PRs
merged in this line are tracked as evolutions under the project-wide
"merged = evolved" convention.

## Why Abra

Abra is the psychic starter — it teleports, it's low HP, it evolves into
one of the highest-raw-intelligence mons in the game. The grant application
is a teleport: minimal effort, right context, land somewhere useful.
Alakazam is what we become after the grant is funded and the agent fleet
has room to grow.
